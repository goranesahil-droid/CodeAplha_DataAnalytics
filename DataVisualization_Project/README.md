# IPL Data Visualization & Analysis 🏏

Exploratory data analysis and visualization of IPL (Indian Premier League) match data, focused on team and player performance comparisons.

## 📊 Dataset

The dataset (`IPL.csv`) contains historical IPL match records used to analyze team and player performance trends across seasons.

## 🎯 Objective

To explore the IPL dataset and uncover insights around:
- Team performance comparisons (wins, losses, consistency)
- Player performance trends (runs, wickets, strike rate, etc.)
- Overall patterns in match outcomes

## 🛠️ Tools & Libraries

- **Python**
- **Pandas** — data cleaning and manipulation
- **Matplotlib** — base plotting
- **Seaborn** — statistical visualizations
