# Zomato EDA — Exploratory Data Analysis

## Dataset
- **zomato.csv** — 9,551 restaurant records across 15 countries
- **Country-Code.xlsx** — Country code mapping

## Tools Used
Python · Pandas · Matplotlib · Seaborn

## Key Findings
- India dominates with 90.6% of all listings
- 22.5% restaurants are unrated (cold-start problem)
- North Indian cuisine is most listed (3,960 restaurants)
- Only India & UAE support online delivery
- Premium restaurants (Price Range 4) rate significantly higher
